# AWB Firmware

Embedded firmware for the Automatic Window Blinds (AWB) running on STM32L4 microcontrollers.

> **Note:** Currently targeting **Nucleo-L476RG** for development. Once the final board (Nucleo-L432KC) arrives, pin definitions will be updated accordingly.
